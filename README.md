## Usage

This modpack uses [packwiz](https://github.com/packwiz/packwiz) to generate its files. 
To export the modpack to an archive file, do the following:

1. Install packwiz on your system
2. Export to curseforge or [modrinth](https://docs.modrinth.com/docs/modpacks/creating_modpacks)
   * `packwiz curseforge export`
   * `packwiz modrinth export`
3. Import the generated archived in your favorite Minecraft launcher
